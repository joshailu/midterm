Poodle Search

This project leveraged HTML and CSS to clone the Google search website. I was able to replicate the search bar, buttons, and layout. Furthermore, options to search for images or perform an advanced search were also available.

Home Page Extension

In this project, I created an extension for Chrome that modifies the new tab page using HTML, CSS, and JavaScript. It overrides the default new tab with a custom page that contains the time, randomly generated quotes, and a Pomodoro timer.

Minesweeper

I used Java and JavaFX to create a program with a graphical user interface to play a Minesweeper game. Boards of various sizes/difficulty can be generated based on the user's preference.
